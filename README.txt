PIF Journal (GitHub Pages upload)

Файлы:
- index.html — журнал (PROD v3)
- .nojekyll — отключает Jekyll обработку на GitHub Pages

Загрузка:
1) Создай публичный репозиторий на GitHub (например pifjournal-site)
2) Upload files -> загрузи эти 2 файла в корень репозитория
3) Settings -> Pages -> Deploy from branch -> main /(root)
4) Custom domain -> pifjournal.com
