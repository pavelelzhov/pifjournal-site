Дашборд

Закрытые сделки считаются по итоговому R, который складывается из частичных фиксаций по TP (доли) + закрытие остатка по STOP/ручному закрытию.
Период: все

Подсказка: клик по строке периода применит фильтр.
Открытые сделки
Пояснение по R (как считаем)
1R = срабатывание первоначального SL на 100% позиции.
Если TP1 закрывает 30% позиции, то вклад в итог: 0.30 × R(TP1).
Если STOP перенесли в БУ/в прибыль — вклад остатка считается по цене STOP, но знаменатель всё равно “первый SL”.

Добавить сделку

Вставь сигнал → “Разобрать” → проверь поля → выставь доли закрытия по тейкам → сохранить.
Парсер понимает форматы A/B/C из примеров (A — фьючи, B — спот, C — фьючи с 4 тейками).
Подсказка: если вставляешь диапазон входа вида (87550-450), мы восстановим 87450 и возьмём среднее (если policy=AVG).

Портянка

Вставляешь несколько сигналов подряд — получаешь предпросмотр и добавляешь пачкой.
Если трейдер/рынок распознался — он победит дефолт.

Настройки

Для PnL≈ в деньгах нужен депозит и риск на сделку. Bybit Live — публичные котировки (без ключей).
WS режим даёт “почти real‑time”. Если WS блокируется сетью — переключись на REST или proxy.

Пресеты долей по трейдерам (если TP больше 1)

Формат: 30,35,35 или 25,25,25,25. Если TP больше, остаток распределим равномерно.
